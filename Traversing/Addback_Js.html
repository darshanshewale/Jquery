Jquery Addback Method

        The Addback is an inbuilt method in Jquery that adds the previous set of elements to the current set
        This method adds previous DOM tree elements to the current set and maintains them  in the internal stack 
        which will take care of changes to the matched set of elements

        .addback(selector)
