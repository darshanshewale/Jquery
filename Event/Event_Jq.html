Jquery_Event

        Event refers to the actions performed by the site visitor during their interactivity with the website (or webpage) There can be various types
        of event such as 
        1)user clicks on the button
        2)user moves mouse pointer over an image
        3)User pressed and key from keyboard etc

        Method Name:
        1)Click:get invoked when the user clicks on the particular HTML element
        2)dbclick: gets invoked when the user double clicks on the particular HTML element
        3)mouseenter:gets invoked when mouse pointer enters the particular HTML element
        4)mouseleave:gets invokes when mouse pointer is removed from the particular HTML element which was selected earlier 
        5)Mousedown:gets invoked when mouse left,right or middle button is pressed while the mouse pointer is over the HTML element
        6)mouseup : gets invoked when mouse left ,right or middle button is released while the mouse pointer is over the HTML element
        7)Hover : gets invoked when mouse pointer enter and leaves the HTML element. It is a combination of mouseenter() and Mouseleave() Method


        
Get and Set Method


        jquery has various method to get the value of an attribute and set the attribute to specific value There method are powerful enough
        to the change the website content and its style 
        1)text()- This method is used to get or set the text content of selected HTML element
        2)html()- This method is used to get or set the content of selected elements 
        3)val()- This method is used to get set the value of various from field in the webpage
        4)attr()-This method is used to get or set the value of various attributes in the webpage
        5)CSS()-This method is used to get or set the value of various CSS properties in the webpage
