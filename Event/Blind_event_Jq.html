Jquery Blind

        The jquery blind() is an inbuilt method that is used to attach one or more event handlers for selected 
        elements and this method specifies a function to run when an event occurs

        $(selector).blind(event,data,function);

        parameter:It accept three parameters that are specified below 
        1)event:This is an event type that is passed to the selected elements
        2)data: This is the data that can be shown over the selected element 
        3)function : this is the function that is performed by the selected elements
