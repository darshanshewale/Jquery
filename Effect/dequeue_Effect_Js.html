Jquery dequeue Method

        The dequeue method is an inbuilt method in jquery that is used to remove the next function from the 
        queue and then it will execute the function . In a queue there will be several functions waiting to run
        dequeue used to remove the top function from the queue and execute that function 

        $(selector).dequeue(name);

        parameter: it accepts a parameter "name" which specifies the name of the queue 
        return value: It returns the selected element that performs the given top function
