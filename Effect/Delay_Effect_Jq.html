Delay Method

        Jquery delay Method is an inbuilt that is used to set a timer to delay the execution of the next item in the queue

        $(selector).delay(para1,para2);
          
        parameter:
        It accept tow parameters which are specified below 

        @para1: It specifies the speed of the delay 
        @para2: It is optional and specifies the name of the queue 

        return value:
        It returns the selected element with specified speed
