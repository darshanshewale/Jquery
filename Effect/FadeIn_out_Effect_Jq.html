FadeIn method

        The fadeIn method in jquery is used to chnage the opacity of selected elements from hidden to visible 
        The hidden elements will not be display

        $(selector).fadeIn(speed,easing,callback)

Fadout method 

        The Fadeout Method in jquery is used to chnage the level of opacity for selected element from visible to hidden
        by using this method the faded element will not occupy any space
        $(selector).fadeIn(speed,easing,callback)

    

        parameter:
        @speed : it is an optional parameter and used to specify the speed of the fading effect. The default value of speedis 400
        millisecond The possible value of speed are (millisecond ,slow, fast)

        @easing: it is an optional parameter and used to specify the speed of element to different point of animation 
        the defualt value of easing is swing The possible value of easing are 
        @swing 
        @linear

        @callback: it is optional parameter The callback function is executed after fadeIn() method is completed
