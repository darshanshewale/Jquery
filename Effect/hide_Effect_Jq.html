Hide Method

        Jquery hide() method is used to hide the selector element it acts like the display:none property of CSS and removes the selected
        element fromt the document 

        $(element_selector).hide(duration,easing,call_function);

        parameter:It accepts three parameters which are specfies below
        @Duration: it specifies the speed of the hide effect
        @Easing: it specifies the speed of the element at different points of animation
        @call_function: This is call-back function to be executed after the hide operation
