Jquery Get Method

        In Jquery.get Method loads data fron the server by using the GET HTTP request. This method returns XMLHttpsRequest Object

        $.get(url,[data],[callback],[type])

        parameter:
        url: String containing the UTL at which request is to be sent 
        data : This is an optional parameter that represents key/value pairs that will be sent to the server.
        callback:This optional parameter represents the function which is to be executed whenever the data is loaded successfully
        type:This parameter represents type of data to be return to the callback function i.e "xml","script","json","html","html","jsonp" or "text"
