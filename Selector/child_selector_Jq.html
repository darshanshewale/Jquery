Jquery : first child selector

            Jquery :first-child selector is used to select every element that is the first child of its parent element
    
            $("selector:first-child");

Jquery Last-child selector 

            Jquery :last-child selector every element that is the last child of it parent

Jquery First-of-type-selector

            Jquery :first-of-type selector is used to select all element that are the first child of a particular type of their parent

            $(":first-of-type")
