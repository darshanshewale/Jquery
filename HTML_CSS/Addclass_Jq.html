Addclass()

    The addclass method is an inbuilt method in jquery that is used to add more properties to each selected element It can also be used
    change the property of the selected element 

    1)By Adding a class name directly
    here the Class name can be used directly with the element which is going to be selected
    $(selector).addclass(classname);

    2)By passing a function to add a new class
    Here A function can be passed to the .addclass() for the selected element
    $(selected).addclass(function);
